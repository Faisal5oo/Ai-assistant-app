# TaskFlow — Task Management & Time Tracking

A premium, distraction-free task and time tracking app built with Next.js 14, Tailwind CSS, Framer Motion, Zustand, and Lucide React.

## Features

- **Dashboard** — Welcome stats, circular time tracker, weekly progress chart, today's timeline, and task checklist
- **Tasks** — Kanban board with create/edit/delete modal
- **Analytics** — Category breakdown, estimated vs actual time, focus score
- **Productivity Hub** — Pomodoro, Time Blocking, Deep Work, Batching, Eisenhower Matrix, Flow State

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS 3.4
- Framer Motion
- Zustand (persisted to localStorage)
- Lucide React
- date-fns

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — production server
